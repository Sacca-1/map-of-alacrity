# Map of Alacrity

Highlights Map of Alacrity shortcut locations on the world map and lets you teleport directly to the selected shortcut.

Location data is sourced from the OSRS Wiki page [Map of Alacrity (Demonic Pacts League)](<https://oldschool.runescape.wiki/w/Map_of_Alacrity_(Demonic_Pacts_League)>).
